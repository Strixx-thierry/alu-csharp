# C# Hello World

This repository contains C# programs for the "Hello World" project. It covers basic concepts such as:

*   Initializing, building, and running C# projects using the `.NET` CLI.
*   Printing text and variables to the console (`Console.WriteLine`).
*   Working with different data types (integers, floats, strings).
*   String concatenation and manipulation.
*   Using format specifiers.
